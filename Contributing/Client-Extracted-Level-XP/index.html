    Client-Extracted Level XP

    Hidden contributor staging page. Source: Assembly-CSharp\FOACJEKBFLG.cs.

    The client stores XP as an integer and computes level through GetLevelAsDouble(xp). Level is capped at 500; XP at or above 235,261,135 returns level 500.0. XP below 540 returns integer level 1.

    Fractional level formula used by the client between level thresholds:

    level = current integer level + (current XP - XP required for current level) / (XP required for next level - XP required for current level)
