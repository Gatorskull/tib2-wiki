# Open World Farming Strategy

Open-world farming strategy changes with ship builds, map access, corporation support, market demand, and balance patches. Treat this page as community guidance, not game rules.

For stable farming rules and references, see [Open World Farming](<../Gameplay/Activities/Player vs Environment (PVE)/Open World Farming.md>), [Pity Loot](../Items/Pity-Loot.md), and [Security Rating](../World/Maps/Security Rating.md).

## Build guidance

- Use enough survivability for the map.
- Add harvesting tools if income is the goal.
- Use [Carrier Leadership](../Ships/Carrier-Leadership.md) once available to move slower farming ships more comfortably.
- Wyrd shield-heavy fleets are repeatedly described in gootchat as strong for farming, but exact shield/capacitor formulas belong on a verified build page.

## Economy guidance

Market values change. Avoid treating old Discord prices as live prices. Ask active players or your corporation, and compare current market listings before deciding what to sell, scrap, or keep.
