# NPC Bust Strategy

NPC bust strategy changes with target type, map risk, caller habits, and current player strength. Treat this page as community guidance, not game rules.

For stable PvE references, see [NPC Busts](<../Gameplay/Activities/Player vs Environment (PVE)/NPC Busts.md>), [Structures](../Entities/Structures/index.md), and [Missions](../Gameplay/Missions.md).

## Group-call patterns

Community notes describe calls such as "planet", "uni", "structure", or "clear" busts. A caller may link:

- A location or structure target.
- A driver ship to follow.
- A minimum survivability expectation.

Use [Common Workflows](../Getting-Started/Common-Workflows.md) for assist/follow basics and [Fleet PvP Rules](<../Gameplay/Activities/Player vs Player (PVP)/Fleet-PvP.md>) for driver concepts that also apply to large PvE groups.

## Cautions

- Do not treat old scrap values as fixed prices.
- Bring enough durability and repair capacity for the target.
- Check whether the target is in PvE space or PvP space before traveling.
- If the bust is in PvP space, normal PvP risks apply unless a daily event disables open-world PvP.
