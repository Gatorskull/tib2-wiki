# Computer Deconstruction Strategy

Computer deconstruction strategy is about timing unlocks. For the reference rules, see [Tech Computers](../Items/Tech-Computers.md).

Every deconstructed CPU advances its own computer category and also advances Victor. Because of that shared Victor progress, deconstructing every computer you find can cause Victor to reach important unlock milestones before the computer category you actually wanted.

## Common Planning Pattern

The cattabliss guide suggests a planning pattern:

- Push the desired computer category close to 1,000 points.
- Stop roughly 20 to 50 points short.
- Use other computers to finish earlier unlocks in the cycle.
- Finish the desired category on the fifth unlock so the bonus Tech II to Tech III kit lands on the computer type you wanted.

Check progress regularly in the player stats menu and in **player name -> Crafting -> Blueprint Unlock Progress**.

## Caveats

- This is community planning guidance, not a game rule.
- The exact best category depends on build goals, current market values, and available computers.
- The current wiki still needs exact Tech Computer stat tables.
