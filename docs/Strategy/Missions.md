# Mission Strategy

Mission strategy changes with player level, map access, corporation activity, and daily objectives. Treat this page as community guidance, not game rules.

For stable mission rules, see [Missions](../Gameplay/Missions.md) and [Outpost Missions](../Gameplay/Activities/Outpost-Missions.md).

## Community notes

- For personal exploration missions, some players leave one sector unexplored in a new map until an "Explore System" mission appears.
- If you will not complete specific personal missions, refreshing up to your daily new-mission limit can help find broader objectives.
- For corporation/alliance missions, avoid keeping extremely specific objectives if the group cannot realistically complete them.
- For universe missions, you cannot control the mission; contribute at least once if you want participation credit.

## Outpost mission notes

Community chat describes outpost mission chains as useful for Tech Points and repair support. Exact outpost chain rewards should stay out of reference pages until verified in game.
