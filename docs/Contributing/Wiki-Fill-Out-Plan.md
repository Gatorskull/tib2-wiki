# Wiki Fill-Out Plan

This old fill-out plan has been retired. It described the original wiki buildout and now contains too much completed or outdated routing information to use as guidance.

Use these current pages instead:

* [Content Standards](Content Standards.md) -- canonical-page rules, source handling, and strategy separation.
* [Wiki Rough-Pass Audit](Wiki-Rough-Pass-Audit.md) -- current risks, remaining data gaps, and cleanup notes.
* [Gootchat Source Review](Gootchat-Source-Review.md) -- notes from `gootchat.txt`.

When adding new content, choose the canonical section first:

* `Ships/` for ships, movement, ship mods, and ship crafting.
* `Items/` for item types, stats, upgrades, Specials, and Tech Computers.
* `Gameplay/` for stable activity rules and ability mechanics.
* `Strategy/` for build advice, tactics, and community guidance.
* `World/` for maps, sectors, security rating, and factions.
