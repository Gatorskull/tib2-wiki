# Structure Warfare Rules

Structure busts are organized PvP attacks against non-NPC corporation defense units: [Planets](../../../Entities/Structures/Planets.md), [Garrisons](../../../Entities/Structures/Garrisons.md), [Outposts](../../../Entities/Structures/Outposts.md), and [Shipyards](../../../Entities/Structures/Shipyards.md).

This page is for game rules and reference logic. For community tactics, see [Structure Busting Strategy](../../../Strategy/Structure-Busting.md).

For shared attack requirements and target rules, see [Combat and Targeting](../../Combat-and-Targeting.md).

## Stable rules

| Rule | Notes |
|------|-------|
| Structure Gear Score | Player defense units need **3,000+ Gear Score** or they are recalled. |
| Attack shield | When a defense unit is attacked, it gets a **90-second immunity shield** so defenders can organize. |
| Break shield | After enough attackers die, a **1-hour protection shield** can be raised. |
| Alliance shield | Alliance leaders can use `:ALLIANCE SHIELD` for a **4-hour shield** once per day. |
| Quick Jump | Players can Quick Jump to some structures under attack. Outpost behavior and alliance-owned edge cases need in-game confirmation. |
| Blocking Quick Jump | Engaging an enemy defense unit may prevent defenders from Quick Jumping to it. Needs confirmation. |

## Skull payout

When a non-NPC structure is destroyed in PvP, attackers earn Skulls based on the structure's Gear Score:

| Structure | Payout |
|-----------|--------|
| Outpost / Shipyard | 1 Skull per 500 Gear Score |
| Garrison | 1 Skull per 400 Gear Score |
| Planet | 1 Skull per 300 Gear Score |

See [Skulls](../../../Entities/Currencies/Skulls.md) for other PvP earning rules.
