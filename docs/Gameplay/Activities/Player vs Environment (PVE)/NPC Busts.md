# NPC Busts

NPC busts are group PvE clears against NPC defense units, planets, or high-value structure targets. Players often use them for loot, credits, missions, and activity rewards.

## Rewards And Progress

| Benefit | Notes |
|---------|-------|
| Loot and scrap value | NPC busts can produce items with scrap value. Exact values are market-sensitive. |
| Mission progress | NPC defense-unit kills can progress some [Missions](../../Missions.md). |
| Reward Points | Some event/reward systems include NPC structure kills. See [Reward Point Gifts](../../../Gameplay/Events/Reward-Point-Gifts.md). |
