# Events

Timed and daily content: free gifts, daily event modifiers, holiday gems, and Party Boxes. Most **reset at 00:00 UTC** each day.

## Daily & Free

| Topic | Description |
|-------|-------------|
| [**Free Daily Gifts**](Free-Daily-Gifts.md) | FREE button at [defense units](/Entities/Structures/index.md), GIFT BOX in menu, weekly Void reward. |
| [**Daily Events**](Daily-Events.md) | 50+ random modifiers (crafting bonuses, small-ship buffs, Love Day / War Day). Type **:DAILY** in chat. |

## Special & Rewards

| Topic | Description |
|-------|-------------|
| [**Holiday Events**](Holiday-Events.md) | Gem events on set dates; kill gem NPCs for rare loot. Clues every 30 minutes. |
| [**Party Boxes**](Party-Boxes.md) | Buffs and activities for corp/alliance; from [Reward Point Gifts](Reward-Point-Gifts.md) or weekly gift. |
| [**Reward Point Gifts**](Reward-Point-Gifts.md) | Earned from [Tech Point](https://www.spellbook.com/purchase/tib2/techpoints.php) purchases, NPC structure kills, PvP. Spend on Party Boxes, Deception Module, skins, and more. |
