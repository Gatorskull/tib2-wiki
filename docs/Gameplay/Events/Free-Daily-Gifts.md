# Free Daily Gifts

Players can return **each day** to claim several rewards. Daily gifts reset at **00:00 Coordinated Universal Time (UTC)** every day.

## How to Claim

| Gift | How |
|------|-----|
| **Crafting resources** | Use the **FREE** button at any [**defense unit**](/Entities/Structures/index.md) (your corp’s or discovered NPC). |
| **GIFT BOX** | **Hamburger menu** (drop-down) → **GIFT BOX**. Randomly upgrades [**ships**](/Ships/index.md) [**Elite Ranks**](/Ships/Elite-Ranks.md), equipped [**item**](/Items/index.md) rarity or quality, durability repairs, and more. |
| **Weekly gift (Void items)** | **Captain Panel → REWARDS → CLAIM WEEKLY GIFT**. Earn powerful [**Void items**](/Items/index.md) by completing weekly [**Party Box**](Party-Boxes.md) missions. |
