# Missions

Each captain can have up to **three captain missions** at a time. Mission categories include **Tutorial Missions**, system-chain missions, shared **Corp Mission**, shared **Alliance Missions**, and one server-wide **Universe Mission**.

## Mission Limits & Deletion

* You can get up to **4 new personal missions per day**, whether by completing missions or deleting them to receive replacements.
* **Corp missions** are progressed by **all corp members**.
* **Alliance missions** are progressed by **all alliance members**.
* **Universe missions** are progressed by all players on the server.
* You **must contribute at least 1 progress** to a shared mission to get its reward.
* **Leaving your corp** resets your shared mission participation (you lose progress toward corp/alliance mission rewards).

Corporations can **cancel** their corp or alliance missions using the **:CORP** and **:ALLIANCE** chat commands.

## Mission Rewards and Refresh

Observed base mission rewards:

| Completion | Base reward |
|------------|-------------|
| Personal, shared, or universal mission | 250 XP and 5 [Tech Points](../Entities/Currencies/Tech Points.md) |
| During **Mission rewards doubled** daily event | 500 XP and 10 Tech Points |

Additional shared-mission rules:

- When a corporation, alliance, or universe mission completes, any player who contributed at least 1 progress gets the full player reward.
- When a corporation or alliance mission completes, any corporation with a contributing player gets double the base reward to the corporation.
- Corp and universe missions refresh once per day. Completing more than one in the same day does not guarantee another new one until the next day.
- Alliance mission availability depends on alliance corporations; each corporation in an alliance can have its own alliance mission once per day.
- A non-personal mission with no progress for 3 days can be automatically cancelled.

!!! note "Needs confirmation"
    Mission reward values should be checked in game after mission-system patches.

## Mission Types

| Type | Description |
|------|-------------|
| Kill Normal NPCs | Standard enemy kills. |
| Kill Elite NPCs | Elite NPCs progress by **1 point per [Elite Rank](../Ships/Elite-Ranks.md)** (e.g. a 3-Star Elite = 3 points). |
| Kill NPC Leaders | Leader kills. |
| Kill NPC Defense Units | Destroy NPC [structures](../Entities/Structures/index.md). |
| Kill Players | Only kills **outside [The Arena](<Activities/Player vs Player (PVP)/Arena.md>)** count, and only when a [**Skull**](../Entities/Currencies/Skulls.md) is earned. **1 point per kill per mission** max. |
| Harvest Resources | Gather from asteroids and nodes. |
| Explore Solar Systems | Visit and explore systems on the [Starmap](../World/Maps/index.md). |
| Deposit Resources | Deposit resources. |
| Craft Item | Craft an item. |
| Upgrade Item | Upgrade an item. |
| Equip Item | Equip an item. |
| Scrap Item | Scrap an item. |
| Mine | Mine an asteroid. |
| Rescue | Rescue crew. |
| Distress | Answer a distress signal. |
| Outpost Deposit | Deposit a specified resource to an outpost. |
| Hunt Named NPC | Kill a named leader. |

## Mission overlap rules

Mission overlap rules:

- Multiple personal missions can progress from the same action. For example, one NPC kill can progress two compatible personal kill missions.
- Multiple overlapping alliance missions do **not** all progress from the same action; the most recently acquired matching mission receives progress.
- Only one corporation mission and one universe mission can be active at a time.

## Important Rules

* **"Aliens" in missions** -- Means **any NPC enemy**, including Pirates. Mission text that says "Aliens" can include Pirates unless specified otherwise.
* **Corporate Defenders** -- [**Corporate Defender**](../Entities/Corporations/Corporate-Defenders.md) kills do **not** count for mission kill credit.
* **Outpost missions** -- Some missions are tied to a specific outpost. Use the mission question-mark details to identify the source. See [Outpost Missions](Activities/Outpost-Missions.md).
