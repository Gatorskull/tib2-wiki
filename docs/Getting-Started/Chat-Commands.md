# Chat Commands

Many TIB2 features can be run from chat. Type **`:HELP`** in game for the full current command list.

!!! note "Command verification"
    Chat commands can change. Treat this page as a quick reference and verify expensive commands in game before using them.

## Common commands

| Command | Use | Notes |
|---------|-----|-------|
| `:HELP` | Opens command/help information. | Best source for the complete current list. |
| `freegift` | Level 25 gift command. | Observed rewards include a Mark upgrade kit, a Victor Mark CPU, and a Skip Drive; you may need to use the code twice to receive the Mark upgrade kit. Consult allies before using or selling the kit. |
| `:ARENA` | Arena access shortcut. | Also available from the player/Captain panel when registration is open. |
| `:corp repair mytp confirm` | Corporation item repair. | Repairs 5 random equipped items by 10% for every online corp member and spends Tech Points. See [Item Repair Quick Start](Quick-Start-Guides/Item-Repair-Quick-Start.md). |
| `:alliance repair mytp confirm` | Alliance item repair. | Repairs 2 random equipped items by 10% for every online alliance member and spends Tech Points. |
| `:CRAFT` | Crafting management. | Used for crafting queue and related item-crafting commands. |
| `:CRAFTVOID` | Void crafting management. | Used by crafters for [Void Items](../Items/Void-Items.md). |

!!! note "Corp repair spelling"
    Some older guide text says `:corp repair my tp confirm`. Current wiki references use `:corp repair mytp confirm`, matching the later repair guide. Verify in game before spending Tech Points.

## Chat quality-of-life

The **R** hotkey can quickly open chat and start a reply to the last player you privately messaged or received a private message from. You can also open chat from the top-left button and use the reply button beside the text entry field.
