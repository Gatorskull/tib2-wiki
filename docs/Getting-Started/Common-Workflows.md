# Common Workflows

Use this page for "where do I click?" basics. Deeper mechanics live on the linked canonical pages.

## First hour

| Goal | Where to go |
|------|-------------|
| Start moving | Click adjacent sectors on the map. See [Sectors](../World/Maps/Sectors.md). |
| Follow the first mission | Open the mission prompt and explore **100% of Sol** while fighting pirates and harvesting asteroids. See [Missions](../Gameplay/Missions.md). |
| Deploy your second ship | After earning another ship, click **Deploy 2nd Ship**. See [Ships](../Ships/index.md). |
| Join a corporation | Open corporation options from the player/captain UI and find an active corp. See [Corporations](../Entities/Corporations/index.md). |

## Settings

| Goal | Path |
|------|------|
| Show sector ship counts | Hamburger menu -> Settings -> Gameplay -> Gameplay Settings -> **Display local sector ship counts**. |
| Show sector names | Hamburger menu -> Settings -> Gameplay -> Gameplay Settings -> **Always show sector name text**. |
| Sort enemies in PvP | Hamburger menu -> **Sort Enemy** -> choose the called sort. |
| Show players over NPCs | Ship button list settings -> **Show player ships over NPC ships**. |

See [Recommended Settings](Settings.md) for why these matter.

## Gifts and rewards

| Goal                                      | Path                                                            |
| ----------------------------------------- | --------------------------------------------------------------- |
| Claim daily crafting resources            | Click **FREE** at a discovered defense unit.                    |
| Claim the daily Gift Box                  | Hamburger menu -> **GIFT BOX**.                                 |
| Claim weekly Void reward                  | Captain Panel -> **REWARDS** -> **CLAIM WEEKLY GIFT**.          |
| Check Reward Points and Party Box bonuses | Captain Panel -> **REWARDS**.                                   |
| Level 25 gift                             | Type `freegift` in chat. See [Chat Commands](Chat-Commands.md). |

See [Free Daily Gifts](../Gameplay/Events/Free-Daily-Gifts.md) and [Reward Point Gifts](../Gameplay/Events/Reward-Point-Gifts.md).

## Items and repairs

| Goal | Path |
|------|------|
| Upgrade rarity or rank | Click item -> **Upgrade**. See [Item Upgrades](../Items/Item-Upgrades.md). |
| Manage item restrictions | See [Item Management](../Items/Item-Management.md) for repair, scrap, deconstruct, auction, and corp-bank rules. |
| Repair an item from the item panel | Click item -> **Upgrade** -> **Repair**. This can reduce quality. |
| Repair through corp/alliance chat | Use `:corp repair mytp confirm` or `:alliance repair mytp confirm`. See [Chat Commands](Chat-Commands.md). |
| Repair ship durability | Click ship -> **REPAIR**. This spends [Skulls](../Entities/Currencies/Skulls.md). |

## Travel and PvP

| Goal | Path |
|------|------|
| Open the Starmap | Hamburger menu -> **Starmap**. |
| Quick Jump | Hamburger menu -> Quick Jump target. Costs 1 [Tech Point](../Entities/Currencies/Tech Points.md). |
| Join Arena | Captain Panel -> **ARENA**, or use `:ARENA` when registration is open. |
| Assist a fleet driver | Click linked driver ship -> **My Assist Leader**, then follow the marked ship. See [Fleet PvP](<../Gameplay/Activities/Player vs Player (PVP)/Fleet-PvP.md>). |
