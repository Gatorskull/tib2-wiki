# Recommended Settings

Some settings make the game easier to read, especially once you join a corporation or start participating in PvP.

## Gameplay settings

Open these from **Hamburger menu -> Settings -> Gameplay -> Gameplay Settings**.

| Setting | Recommendation | Why it matters |
|---------|----------------|----------------|
| **Display local sector ship counts** | On | Shows local friendly, neutral, PvP enemy, and NPC counts. This helps prevent [collision](../World/Maps/Sectors.md#collision-stacking-limit) and helps read enemy presence. |
| **Always show sector name text** | On | Makes sector coordinates visible on the main screen, which helps with mission targets, boss locations, and fleet calls. |
| **UTC time/local time** | Player preference | The game operates on UTC. Daily reset is at 00:00 UTC, and event rolls commonly use UTC times. |
| **Camera snap back delay** | Player preference | Controls how quickly the camera returns to your ships after you scroll away on the map. |
| **Top down/Side View** | Player preference | Lets you choose between a flatter map view and a more angled view. |

## Ship button list

Open these from the ship button list settings.

| Setting | Recommendation | Why it matters |
|---------|----------------|----------------|
| **Show player ships over NPC ships** | On | Especially useful in PvP because player targets stay visible when mixed with NPCs. |
| **Sort allied ships most in danger** | On | Helps repair/support pilots find endangered friendly ships. |
| **Ship Buttons -> show buttons** | Usually always, except resources | Keeps common actions visible without extra clicking. |
