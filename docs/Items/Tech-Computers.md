# Tech Computers

**Tech Computers** are upgraded computer items created through computer deconstruction progress. They have three tiers: **Tech I**, **Tech II**, and **Tech III**. Higher tiers grant stronger positive stats and Equip Point bonuses compared with similar MK computers.

Planning advice belongs in [Computer Deconstruction Strategy](../Strategy/Computer-Deconstruction.md) or the cattabliss [Computer Deconstruction Quick Start](../Getting-Started/Quick-Start-Guides/Computer-Deconstruction-Quick-Start.md).

## Deconstruction Progress

Computer deconstruction grants progress toward upgrade kits:

| Rule | Detail |
|------|------------------------------|
| Progress target | 1,000 points in a computer category unlocks a kit. |
| Category progress | Deconstructing a computer grants points toward that computer's category. |
| Victor progress | Deconstructing any CPU also grants equal progress toward the Victor computer category. |
| Points by rarity | Common grants 1 point, Uncommon 2, Rare 3, and so on by rarity rank. |
| Progress location | Player name -> Crafting -> Blueprint Unlock Progress. |

## Kit Cadence

| Unlock count | Reward |
|--------------|--------|
| Unlocks 1-4 in a cycle | A kit that upgrades a computer from Tech I to Tech II. |
| Every 5th unlock | The Tech I to Tech II kit plus a kit that upgrades Tech II to Tech III. |

The fifth unlock matters because the extra Tech II to Tech III kit follows the computer category that reaches the milestone.

General restrictions on whether an item can be deconstructed are covered on [Item Management](Item-Management.md).
