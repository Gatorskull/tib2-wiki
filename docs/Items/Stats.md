# Stats

Ships, [defense units](/Entities/Structures/index.md), and [items](index.md) provide **stats** that affect combat, movement, and abilities. Stats from items are subject to [**Diminishing Returns**](Diminishing-Returns.md).

## Stat List

| Stat | Effect |
|------|--------|
| **Resource Capacity** | Max resources a ship can carry per resource type. |
| **Move Speed** | Ship movement speed as a percentage of base speed. |
| **Hull Repair** | Amount repaired with outgoing hull repair actions. |
| **Outgoing Damage Bonus** | Overall damage output (weapons, drones, battle ram, etc.). |
| **Outgoing Hull Damage** | Damage dealt to enemy hull. |
| **Shield Damage** | Damage dealt to enemy shields. Excess rolls over to hull. |
| **Shield Pierce** | Percentage of attack damage vs shields that passes to hull. |
| **Incoming Damage** | Amount of damage the unit receives from attacks. |
| **Xp Bonus** | XP earned from NPC and PvP kills (Ship, Player, Corporation XP). |
| **Critical Damage** | Damage dealt when a critical hit occurs. |
| **Critical Chance** | Chance to land a critical hit. Counters Critical Resist. |
| **Critical Resist** | Reduces chance of receiving critical hits. |
| **Hit Chance** | Accuracy; reduces GRAZE chance. Counters Evasion. Benefits hotbar abilities (Grapple, Hack). |
| **Evasion Chance** | Chance to convert incoming attacks into a GRAZE. Counters Hit Chance. |
| **Drain Power** | Strength of the Drain hotbar ability; counters Drain Resist. |
| **Drain Resist** | Counters the Drain hotbar ability. |
| **Grapple Power** | Grapple hotbar ability duration and strength; counters Grapple Resist. |
| **Grapple Resist** | Counters the Grapple hotbar ability. |
| **Hack Power** | Hack hotbar ability duration and strength; counters Hack Resist. |
| **Hack Resist** | Counters the Hack hotbar ability. |
| **Splash Chance** | Chance that attacks deal area-of-effect damage. Counters Splash Resist. |
| **Splash Resist** | Reduces or negates AoE damage. Also benefits vs Nova and Garrison Strike. |
| **Power Recharge** | Battery recharge rate. |
| **Power Capacity** | Max battery capacity. |
| **Max Hull** | Total hull of the unit. |
| **Shield Capacity** | Total shields of the unit. |
| **Targeting Speed** | Speed at which weapons and drones lock onto targets. |
