# Security Rating

**Security Rating (SR)** describes how dangerous a map is. Higher-SR maps have stronger enemies, greater travel risk, and better reward potential.

Use Security Rating as a danger and reward signal, not as a complete build recommendation. Ship counts, recommended player levels, and recommended hull tiers still need live observation.

## Security Rating Table

| Security rating | Display label | NPC level band | Reward and rule notes |
|-----------------|---------------|----------------|-----------------------|
| **SR1** | Safe Zone / Security Safe Zone | 2-8 | Starter-safe area. |
| **SR2** | Security 2 | 7-13 | Higher NPC levels than SR1. |
| **SR3** | Security 3 | 13-23 | Ultra-Rare pity progress becomes more meaningful around this point. |
| **SR4** | Security 4 | 23-35 | Higher NPC levels and risk. |
| **SR5** | Security 5 | 35-50 | Higher NPC levels and risk. |
| **SR6** | Security 6 | 50-80 | Higher NPC levels and risk. |
| **SR7** | Security 7 | 80-110 | Higher NPC levels and risk. |
| **SR8** | Security 8 | 110-130 | 25% loot boost; SR8 capture has alliance-citizen rules. |
| **SR9** | Security MAX | 130-150 | Homeworld Raids give double pity loot. |

Special encounters can use their own level bands:

| Encounter type | Level band |
|----------------|------------|
| Boss | 140-160 |
| Faction Boss | 150-180 |
| Instance | 100-130 |

## What Security Rating Changes

- Enemies become more difficult in higher-SR maps.
- Pity loot progresses faster in more dangerous areas.
- Ultra-Rare pity progress is limited in lower-risk areas and becomes meaningful around **SR3**.
- **SR8** maps receive a **25% loot boost**.
- **SR9 Homeworld Raids** receive **double pity loot**.
- Some alliance mechanics specifically reference **SR8** map capture.

## Missing Live Data

These fields should be filled only from in-game observation, patch notes, or another named source.

| Missing field | What to collect |
|---------------|-----------------|
| NPC ships per sector | Observed max or common range for each SR. |
| Recommended player level | Practical level range for hunting or traveling in each SR. |
| Recommended ship/hull tier | Practical minimum hull, shield, or ship class guidance. |
| Practical danger notes | Common reasons players die or avoid each SR. |

If a value is a best guess, label it as **Estimated** instead of presenting it as a rule.
