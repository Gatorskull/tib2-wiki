# Equip Points

**Equip Points** are the capacity your ship or defense structure has for mounting weapons, armor, shields, batteries, drones, and other equipment. [Larger ships and defense units](Size-And-Targeting.md) have more Equip Points.

The sum of all equipped item costs cannot exceed the unit's maximum Equip Points.

---

## What changes Equip Point cost

Most equipment has an Equip Point cost. That cost can change when the item is upgraded:

- Changing the **rank** of weapons, armor, storage, or shields changes both the item's power and its Equip Point cost. See [Item Upgrades](../Items/Item-Upgrades.md).
- Increasing an item's **rarity** can reduce its requirements.
- Some rare bonuses and ship upgrades can add extra Equip Points.

---

## Ship Equip Point formula

Ship Equip Points are calculated from several sources:

| Source | Effect |
|--------|--------|
| Ship class and faction | Provides the base Equip Point value. |
| Hardcore ship | Adds +3 Equip Points. |
| Elite rank, non-Striker ships | Adds +1 Equip Point per Elite Rank. |
| Elite rank, Striker Gunboat | Uses a special formula: +2 plus +3 per Elite Rank. |
| Extra Striker flag | Adds +3 more Equip Points. Public trigger still needs confirmation. |
| Ship mod bonus | Adds any bonus Equip Points from ship mods. |
| `EquipPointMax` bonus stat | Adds the bonus-stat value to total Equip Points. |

Some special units can have 50 Equip Points. Do not use that as a normal ship rule unless the specific unit is confirmed.

---

## Crafted ship bonuses

There is a **1% chance** that a [crafted ship](Ship-Crafting.md) gains **+1 Equip Point**. In very rare cases, that crafted bonus can be **+2** or **+3**.
